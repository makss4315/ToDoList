# 🗂️ To-Do List (JavaFX Task Manager)
A desktop To-Do List / Task Manager application built with Java and JavaFX.
The application allows users to create, edit, move, and manage tasks using a Kanban-style board interface.
---

## 🛠️ Technologies Used
- Java
- JavaFX
- Custom UI styling (JavaFX CSS styles)

## ▶️ How to Run
## 1️⃣ Requirements

- Java 17+ (recommended)

- JavaFX SDK

## 2️⃣ Clone repository

